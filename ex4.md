# No.4 ◯年は何秒

## 問題文

次のプログラムをコピー＆ペーストして、指定した場所にプログラムを追記することで問題を解いて下さい。

```fortran
program main
    implicit none
    integer(4):: seconds = 365*24*60*60 ! ←一年の秒数

    ! 以下のコメント!~~ を消して追記する
    print*, !1年は何秒か
    print*, !2年は何秒か
    print*, !5年は何秒か
    print*, !10年は何秒か
end program
```

integer型の変数`seconds`は一年の秒数を表しています。これを利用して

* 1年は何秒か
* 2年は何秒か
* 5年は何秒か
* 10年は何秒か
  
を順に一行ずつ表示するプログラムを作って下さい。
うるう秒やうるう年のことは考え無くて良いとします。
